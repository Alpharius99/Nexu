## Status Legend

| Status | Meaning |
|--------|---------|
| OPEN | Bug confirmed, not yet fixed |
| IN PROGRESS | Fix being worked on |
| FIXED | Fix merged to main |
| WON'T FIX | Accepted behavior or out of scope |
| DUPLICATE | Same as another bug |

## Severity Legend

| Severity | Meaning |
|----------|---------|
| P0 — Critical | App crashes, data loss, corruption |
| P1 — High | Feature broken, no workaround |
| P2 — Medium | Feature broken, workaround exists |
| P3 — Low | Cosmetic, minor inconvenience |
