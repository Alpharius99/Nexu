# Nexu — Mistakes Log

> Track mistakes made during development so we learn from them and don't repeat them.

---

## How to Use This Document

When a mistake is discovered — wrong approach, bad assumption, wasted effort, regression — log it here with:
- **What happened** (the symptom)
- **Root cause** (why it happened)
- **Fix applied** (what we did)
- **Lesson** (how to avoid it next time)
